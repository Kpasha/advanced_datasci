    \section{Introduction}\label{introduction}

    The \href{https://www.zillow.com/promo/Zillow-prize/}{Zillow Prize} is a
\href{https://www.kaggle.com/c/zillow-prize-1}{Kaggle competition} that
aims to inspire data scientists around the world to improve the accuracy
of the Zillow "Zestimate" statistical and machine learning models.

My goal is to compete for the Zillow prize and write up my results.

    \section{Methods}\label{methods}

    \subsection{Data}\label{data}

    The data were obtained from
\href{https://www.kaggle.com/c/zillow-prize-1/data}{Kaggle website} and
consist of the following files:

\begin{itemize}
\tightlist
\item
  properties\_2016.csv.zip
\item
  properties\_2017.csv.zip
\item
  sample\_submission.csv
\item
  train\_2016\_v2.csv.zip
\item
  train\_2017.csv.zip
\item
  zillow\_data\_dictionary.xlsx
\end{itemize}

The \texttt{zillow\_data\_dictionary.xlsx} is a code book that explains
the data. This data will be made available on
\href{https://figshare.com/}{figshare} to provide an additional source
if the \href{https://www.kaggle.com/c/zillow-prize-1/data}{Kaggle site
data} become unavailable.

    \subsubsection{Reproducibility}\label{reproducibility}

    Reproducibility is extremely important in scientific research yet many
examples of problematic studies exist in the literature (Couzin-Frankel
2010)\cite{5251998/UXR4ZTUS}.

accompanying \texttt{env.yml} file in the \texttt{config} folder. The
computational environment used to analyze the data can be recreated

\href{https://conda.io/docs/using/envs.html}{\texttt{conda} package and

the analysis is included in the \texttt{README.md} file in the
\texttt{config} folder.

(\texttt{01\_zillow\_MWS.ipynb}) or Python script
(\texttt{01\_zillow\_MWS.py}), can also be run on the Kaggle website
